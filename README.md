# AWS-EC2-LEMP-Instagram-Login-Application-Deployment

## Scenario
Your company wants to deploy an Instagram-style login application on AWS using
the LEMP stack.You have been provided with the frontend and backend application code. Your
task is to deploy the application on an EC2 instance, configure the database, run
the application on a custom port, and later migrate the application data to another
EC2 server using an EBS volume.
